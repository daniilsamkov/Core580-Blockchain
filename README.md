# CORE580 Blockchain 🚀

A decentralized Python blockchain network running from a home server. Optimized for SHA-256 GPU mining simulation.

## How to join and mine:
1. Install required libraries:
pip install -r requirements.txt

2. Open `core580_wallet.py` and replace `NODE_URL` with the active tunnel link.
3. Run the miner:
python core580_wallet.py
Sorry for not having full information yet, the coin and miner are still in development
